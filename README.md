# Hanatour Design System for Vibe-Coding 🚀

이 프로젝트는 개발 지식이 깊지 않은 **기획자, 디자이너, 바이브코더(Vibe-coder)**들이 AI(Cursor, Claude 등)를 활용해 하나투어 브랜드 규격에 맞는 웹/앱 화면을 **명령 한 마디로 순식간에 찍어낼 수 있도록 고안된 AI-Native 디자인 시스템**입니다.

복잡하게 코드를 복사하거나 임포트(Import)할 필요가 전혀 없습니다. AI에게 이 시스템(스킬)을 쥐여주면, 알아서 브랜드 컬러, 폰트, 간격, 사내 규칙을 지켜 완벽한 페이지를 직조해 냅니다.

---

## 🧐 언제 사용해야 하나요?

### 🎯 **새로운 사내 프로덕트(MVP)를 기획하고 빠르게 눈으로 보고 싶을 때**
초안을 그리기만 하고 끝나는 것이 아니라, 실제 웹 위에서 동작하는 하나투어 톤앤매너의 제품(프로토타입)을 하루 만에 뽑아내고 싶을 때 매우 유용합니다.

### 🎨 **버튼 색상, 폰트 크기, 여백을 일일이 지정하기 귀찮을 때**
기획자가 AI에게 "이 버튼은 보라색, 둥글게 해줘" 하고 구구절절 쓸 필요가 없습니다. 하나투어 스킬을 켜두면, 알아서 브랜드 표준 규격에 맞춰 제작됩니다.

### 🚀 **외주나 개발자 없이 아이디어를 실체화하고 싶을 때**
"아이디어는 있는데 당장 만들어줄 개발자가 없네" 하는 상황에, AI에게 스킬을 쥐어주고 프롬프트만 치면 바로 작동하는 결과물이 나오므로 소통 비용이 압도적으로 줄어듭니다.

---

## 🏛 1. 시스템 아키텍처 (어떻게 작동하나요?)

이 디자인 시스템은 단순한 이미지 쪼가리가 아닙니다. 철저히 **AI가 코드를 짜기 좋도록** 설계된 3단 구조로 이루어져 있습니다.

1. **디자인 토큰 (Design Tokens)**: `tailwind.config.js`
   - 하나투어의 시그니처 색상 (Master Purple, Accent Yellow 등)과 라운딩, 그림자 값이 AI가 이해할 수 있는 수학적 변수로 정의되어 있습니다.
   - *결과:* AI가 절대 엉뚱한 색(예: 이상한 칙칙한 보라색)을 사용하지 못하도록 강제합니다.
2. **사전 조립된 리스폰시브 컴포넌트 (UI Components)**: `hanatour-components/`
   - 버튼, 배지, 카드, 모달 등 자주 쓰이는 블록들이 이미 코드로 짜여 있습니다.
   - *결과:* AI가 밑바닥부터 버튼을 만들며 시간을 낭비하거나 디자인을 흩트리지 않고, 준비된 고퀄리티 블록을 가져와 레고처럼 조립합니다.
3. **AI 전용 행동 수칙 (Agent Skill)**: `ai-skill/`
   - AI가 위 두 가지(토큰과 컴포넌트)를 언제 어떻게 써야 하는지 적어둔 세뇌(?) 명령어 폴더입니다.
   - *결과:* 사용자가 임포트를 하거나 구조를 잡을 필요 없이, 시작 버튼부터 끝 버튼까지 AI가 백그라운드 환경을 지배합니다.

---

## 📂 2. 파일 및 폴더 구조 안내

기획자가 직접 코드를 고칠 일은 적지만, 패키지가 어떻게 구성되어 있는지 참고하실 수 있습니다.

| 파일/폴더명 | 설명 | 비고 |
|-----------|------|------|
| **`ai-skill/`** | 본인의 AI 툴 에디터 안에 장착할 "하나투어 전용 AI 스킬" 폴더 | ⭐️ 마법의 코어 |
| **`hanatour-design-system.md`** | AI가 UI를 만들 때 어떻게 색칠해야 하는지 읽고 학습하는 시각 가이드 문서 | 중요 가이드 |
| **`tailwind.config.js`** | 브랜드 컬러, 간격 등이 변수로 저장된 디자인 시스템 치트키 파일 | 설정 파일 |
| **`hanatour-components/`** | 버튼, 카드 등 AI가 조립할 때 꺼내 쓰는 준비된 React(TSX) 레고 블록 | 컴포넌트 베이스 |
| `hanatour-tokens.json` | 순수 W3C 포맷 시스템 기반 데이터 원본 | 참고용 |

## 🤖 3. 바이브 코딩 (어떻게 사용하나요?)

일반 개발 방식과 다르게 파일 설치나 코드 복사, 환경설정은 기획자가 신경 쓸 몫이 아닙니다. 철저하게 **말(Prompt)**로만 조종하세요.

### Step 1. AI에게 스킬 장착하기
이 레포지토리 안에 들어있는 `ai-skill/hanatour-design-system` 폴더를 복사하여, 사용 중인 AI 툴(Claude Code, Cursor 등)의 스킬 폴더(예: `~/.gemini/antigravity/skills/` 등)에 넣기만 하세요. 한 번 세팅하면 끝납니다.

### Step 2. AI에게 명령 내리기 (바이브 코딩)
VS Code나 모니터 앞에 앉아 텅 빈 폴더를 열고, AI 챗봇에게 아래와 같이 자유롭게 명령하세요.

> **🎤 Prompt 예시:**  
> "하나투어 가이드라인 스킬 써줘. 그리고 메인 프로모션 페이지 하나 만들어볼래? 상단에는 하와이 프로모션 배너가 크게 들어가고, 그 밑에는 추천 여행지 카드 4개가 가로로 스크롤 되게 만들어줘. 버튼은 전부 '예약하기'로 하고 눈에 띄게 하나투어 포인트 컬러로 잡아줘."

### Step 3. 커피 마시며 구경하기
AI가 백그라운드에서 다음 작업을 **모두 알아서** 수행합니다.
- 프론트엔드 환경 감지 및 셋업 (Tailwind 세팅 등)
- 시스템 컴포넌트 파일들 자동 다운로드 및 복사
- 최상단 `import` 구문 자동 작성
- **완벽한 하나투어 브랜드 피부가 입혀진 페이지 렌더링**

🔥 **이제 코드는 AI에게 맡기고, 기획과 비즈니스 로직에만 집중하세요!**
